# Personas — pith doing real jobs

Each subdirectory is a persona *actually using pith to do their job* on real public data — not
a synthetic demo. Each holds the persona's deliverable + a short write-up of which pith calls
they used and where the SDK helped or got in the way (DX feedback from the field).

| Directory | Persona | Job |
|---|---|---|
| `sdr/` | SDR / sales rep | Build + enrich a target company list |
| `agency-founder/` | Solo web-agency founder | Find dated-site small businesses + owner contact + outreach angle |
| `investigator/` | OSINT investigator | Identity dossier on a public figure from a thin fact |
| `journalist/` | Journalist | Verify a source + find a contact channel, with provenance |
| `recruiter/` | Technical recruiter | Enrich a candidate from a GitHub handle |
| `security-researcher/` | Security researcher (defensive) | Public-footprint recon on an authorized domain |

All public data, deterministic, no LLM in pith. See `../README.md` for the SDK basics.
